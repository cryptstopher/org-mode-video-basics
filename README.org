* What is GTD?

- /Getting Things Done/ by David Allen
- Systematic approach to productivity and "time management"
  - You cannot manage time but *actions*
- GTD consists of...
  - Calendar
  - Various lists
    - Next actions by context
    - Projects
    - "Someday"
    - Getting creative
